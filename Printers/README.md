Problem 1: Printer shows as “Offline”

1. Check power — Ensure the printer is turned on and not in sleep mode.
2. Check network connection — Verify the printer is connected to Wi-Fi or Ethernet.
3. Restart the printer — Clears temporary hardware or network issues.
4. Restart the computer — Refreshes the connection between device and printer.
5. Ping the printer — Confirms the printer is reachable on the network.
6. Verify IP address — Ensure the printer IP matches system configuration.
7. Check printer port — Confirm correct TCP/IP port is selected in settings.
8. Disable “Use printer offline” — Ensures Windows is not forcing offline mode.
9. Reinstall printer — Fixes corrupted configurations or connections.
10. Update/reinstall driver — Resolves compatibility or driver-related issues.
