Problem 1: Computer is slow

1. Restart the computer — This clears temporary processes and memory leaks that slow down performance.
2. Open Task Manager — Identifies which applications are consuming high CPU, RAM, or disk usage.
3. Close unnecessary programs — Reduces system load by freeing up resources.
4. Disable startup programs — Prevents unnecessary apps from launching at boot and slowing the system.
5. Run Disk Cleanup — Removes temporary files that clutter the system and reduce performance.
6. Check disk usage — High disk usage (100%) can indicate system bottlenecks.
7. Run antivirus scan — Detects malware that may be consuming system resources.
8. Update Windows — Fixes bugs and improves performance through patches.
9. Test with a new user profile — Determines if the issue is related to a corrupted user profile.
10. Check Event Viewer — Identifies system errors or warnings affecting performance.

Problem 2: Application will not start

1. Restart the computer — Clears temporary errors preventing the application from launching.
2. Run as administrator — Ensures the app has the required permissions to start.
3. Check Task Manager — Confirms the app isn’t already running in the background.
4. Verify installation — Ensures the application is properly installed.
5. Repair the application — Fixes corrupted or missing files.
6. Reinstall the application — Replaces damaged files completely.
7. Check antivirus blocking — Security software may prevent execution.
8. Check compatibility mode — Ensures the app is compatible with the OS version.
9. Review Event Viewer logs — Helps identify the exact failure reason.
10. Test with another user account — Determines if the issue is profile-specific.

Problem 3: Computer freezes or hangs

1. Wait briefly — Temporary freezes may resolve if the system is processing tasks.
2. Open Task Manager — Identifies unresponsive applications.
3. End unresponsive processes — Frees system resources by stopping stuck apps.
4. Restart the computer — Clears the system state and resets processes.
5. Disconnect external devices — Faulty peripherals can cause system instability.
6. Check RAM usage — High memory usage can lead to freezing.
7. Update drivers — Outdated drivers often cause instability.
8. Boot in Safe Mode — Helps determine if the issue is software-related.
9. Check disk health — Disk errors can cause system freezes.
10. Review system logs — Identifies underlying issues causing crashes.

---

Problem 4: No display on screen

1. Check power connection — Ensures the monitor and PC are receiving power.
2. Check display cables — Loose or faulty cables can prevent signal transmission.
3. Test another monitor — Helps determine if the issue is with the screen.
4. Use Win+P — Switches display modes in Windows.
5. Restart the computer — Resets display initialization.
6. Reset graphics driver (Win+Ctrl+Shift+B) — Restarts the GPU driver.
7. Check GPU in Device Manager — Ensures graphics hardware is recognized.
8. Update graphics drivers — Fixes compatibility or driver issues.
9. Test another port (HDMI/DP) — Rules out port failure.
10. Check BIOS display — Determines if issue is hardware-level.

Problem 8: Disk is full

1. Run Disk Cleanup — Removes temporary files and system junk.
2. Delete unnecessary files — Frees up storage space.
3. Empty Recycle Bin — Permanently removes deleted files.
4. Uninstall unused programs — Frees significant disk space.
5. Move files to cloud storage — Offloads data to external storage.
6. Check Downloads folder — Often contains large unused files.
7. Remove old Windows updates — Cleans system update leftovers.
8. Identify large files — Finds storage-heavy items.
9. Enable storage sense — Automates cleanup tasks.
10. Compress files — Reduces disk usage.

Problem 10: System crashes (blue screen / restart)

1. Restart the computer — Temporary crashes may not reoccur.
2. Note error message — Provides clues for troubleshooting.
3. Update Windows — Fixes known system bugs.
4. Update drivers — Outdated drivers often cause crashes.
5. Remove recent software — New installations may cause conflicts.
6. Run malware scan — Malware can destabilize the system.
7. Check RAM (memory test) — Faulty RAM causes crashes.
8. Check disk errors — Corrupt disks lead to system failure.
9. Boot in Safe Mode — Helps isolate the issue.
10. Review crash logs — Identifies root cause for escalation.
