Problem 1: Cannot log in to Microsoft 365

1. Check username and password — Ensure the user is entering correct credentials without typos.
2. Check Caps Lock and keyboard layout — Prevent incorrect input due to keyboard settings.
3. Try login via browser — Confirms whether the issue is app-specific.
4. Reset password — Fixes access issues caused by incorrect credentials.
5. Check MFA — Ensure multi-factor authentication is completed correctly.
6. Verify internet connection — Login requires stable connectivity.
7. Clear browser cache and cookies — Removes corrupted session data.
8. Try another browser or device — Helps isolate environment-specific issues.
9. Check account status — Ensure account is not locked or disabled.
10. Check Microsoft service status — Determines if the issue is a global outage.

Problem 2: Outlook does not open

1. Restart the computer — Clears temporary system issues.
2. Run Outlook in Safe Mode (`outlook /safe`) — Starts without add-ins.
3. Disable add-ins — Faulty add-ins often cause crashes.
4. Check Task Manager — Ensure Outlook is not running in background.
5. Create new Outlook profile — Fixes profile corruption.
6. Repair Office installation — Restores missing or damaged files.
7. Check OST/PST files — Corrupt data files can block startup.
8. Update Office — Fixes bugs and compatibility issues.
9. Check Event Viewer logs — Identifies specific application errors.
10. Reinstall Office — Resolves deep corruption issues.
